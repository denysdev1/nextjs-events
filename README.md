## Run Locally

Clone the project

```bash
  git clone git@github.com:denysdev1/nextjs-events.git
```

Go to the project directory

```bash
  cd nextjs-events
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
